## Features
- Basic operations: `+ - * / %`
- Exponentiation: `^`
- Square root: `sqrt()`
- Trigonometry: `sin()`, `cos()`, `tan()`, `asin()`, `acos()`, `atan()` (degree/radian toggle)
- Hyperbolic functions: `sinh()`, `cosh()`, `tanh()`, `asinh()`, `acosh()`, `atanh()`
- Logarithms: `log()`, `ln()`, `logn(base, x)`
- Exponential: `exp()`
- Factorial: `fact()`
- Constants: `pi`, `e`
- Operator precedence & parentheses
- History with timestamp
